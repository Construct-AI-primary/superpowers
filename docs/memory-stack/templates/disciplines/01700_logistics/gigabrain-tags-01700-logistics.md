### For Logistics Tasks
1. **Start with Primary Tags**: Use `disciplines, 01700_logistics` as base
2. **Add Process Tags**: Include specific process tags (supply-chain, transportation, etc.)
3. **Include Cross-Discipline**: Add related discipline tags when relevant
4. **Use Technical Tags**: Include implementation tags for development tasks

### Search Strategies
1. **Broad Search**: `disciplines, 01700_logistics` + 1-2 process tags
2. **Specific Search**: Add more specific tags for targeted results
3. **Cross-Discipline**: Include related discipline tags for integration tasks
4. **Technical Focus**: Use technical tags for implementation details
