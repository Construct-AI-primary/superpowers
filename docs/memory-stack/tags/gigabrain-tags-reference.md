## Search Strategy Guide

### 1. Start with Content Type
- Need documentation? → `guides, codebase, documentation`
- Need procedures? → `procedures, [area], codebase`
- Need analysis? → `analysis, codebase, research`

### 2. Add Domain Specificity
- Security focus → `+ security, codebase, compliance`
- Database work → `+ database, codebase, data-management`
- Testing → `+ procedures, testing-qa, codebase`

### 3. Include Discipline Context (when relevant)
- Procurement work → `+ disciplines, 01900_procurement`
- Safety compliance → `+ disciplines, 02400_safety`

### 4. Use 2-3 Tags Maximum
- More tags = fewer results
- Focus on most relevant combinations
- Test and refine based on results
