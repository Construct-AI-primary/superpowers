# InfraForge AI Skills

## Overview

InfraForge AI specializes in infrastructure automation, DevOps practices, and cloud operations powered by AI. This directory contains skills for building, deploying, and managing scalable infrastructure.

## Focus Areas

- **Infrastructure as Code** - AI-driven IaC generation and optimization
- **Cloud Operations** - Multi-cloud management and orchestration
- **CI/CD Automation** - Intelligent pipeline creation and optimization
- **Monitoring & Observability** - AI-powered alerting and diagnostics

## Skill Categories

### Cloud Infrastructure
- AWS, Azure, GCP resource management
- Kubernetes cluster operations
- Serverless architecture deployment

### DevOps Automation
- Pipeline generation and optimization
- Container orchestration
- Configuration management

### Security & Compliance
- Infrastructure security scanning
- Compliance automation
- Secret management

### Performance Optimization
- Cost optimization recommendations
- Resource scaling strategies
- Performance tuning

## Getting Started

Skills in this folder are specific to InfraForge AI implementations. For cross-platform skills, see the `shared/` directory.
